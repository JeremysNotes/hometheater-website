# Device Basics
**ht-pdu01** is a network connected PDU that provides remotely switchable 110VAC power to the core A/V gear including the crosspoint matrixes and scaler devices. The PDU hosts 8 switchable outlets and 2 always-on outlets, along with a wired 100mbps ethernet port. Devices plugged into this will only be powered on when necessary to ensure only the required devices for a specific scenario are turned on. based input from the home theater touchscreens
